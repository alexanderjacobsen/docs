This guide shows the smallest viable Mantl integration: install the SDK, register a project, wire up auth, drop the sidebar into your app.

## 1. Install the SDK

```bash
npm install @mantl/react
```

## 2. Create a project

Sign in to the [Mantl dashboard](https://app.mantl.dev), create a project, and generate an API key. The key looks like `sk_proj_<projectId>_<secret>` and is shown **once** — save it immediately.

## 3. Add a token endpoint

The SDK never sees your project's secret key. Instead, your server signs a short-lived JWT and the SDK fetches that. In Next.js this is one route:

```ts app/api/token/route.ts
import { SignJWT } from "jose";
import { NextResponse } from "next/server";

const SECRET = process.env.MANTL_SECRET_KEY!;

export async function GET() {
  const projectId = SECRET.match(/^sk_proj_([a-z0-9-]+?)_/)![1];

  const token = await new SignJWT({
    sub: "user-123",       // your real user id goes here
    proj: projectId,
    name: "Alice",
  })
    .setProtectedHeader({ alg: "HS256" })
    .setIssuedAt()
    .setExpirationTime("24h")
    .sign(new TextEncoder().encode(SECRET));

  return NextResponse.json({ token });
}
```

Add `MANTL_SECRET_KEY` to your `.env.local`.

<Tip>
  The `sub` claim is your application's user id. Mantl uses it to scope conversations and preferences per user — if you change it across sessions, history won't carry over.
</Tip>

## 4. Drop in the sidebar

```tsx app/page.tsx
"use client";

import { MantlProvider, MantlSidebar } from "@mantl/react";

export default function Home() {
  return (
    <MantlProvider tokenUrl="/api/token">
      <main>
        {/* your app */}
      </main>
      <MantlSidebar position="right" />
    </MantlProvider>
  );
}
```

That's it. Open the page, click the sparkle launcher in the corner, and chat.

## 5. Add a tool (optional)

A chat without tools is just an LLM. To make it useful, give it tools that read or modify your app:

```tsx
import { z } from "zod";

const tools = {
  navigateTo: {
    description: "Navigate the user to a page in the app",
    parameters: z.object({
      path: z.string().describe("The path to navigate to, e.g. /orders/42"),
    }),
    execute: async ({ path }: { path: string }) => {
      window.location.href = path;
      return { success: true };
    },
  },
};

<MantlProvider tokenUrl="/api/token" tools={tools}>
  ...
</MantlProvider>
```

The AI can now call `navigateTo` whenever it judges the user wants to go somewhere.
